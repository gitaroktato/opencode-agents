# OpenCode Agentic Configs

## Installation

Copy over to your `~/.config/opencode` folder using:

```bash
cp -TRf .opencode/agents ~/.config/opencode/agents
cp -TRf .opencode/skills ~/.config/opencode/skills
cp ./opencode.jsonc ~/.config/opencode
```
